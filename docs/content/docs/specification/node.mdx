---
title: Node
description: Node that can be used to create custom functionality in the system.
---

## Specification

```yaml
kind: Node
name: string
func: string
description: Optional[LanguageBundle]
prompts: Optional[LanguageBundle]
nodes: (Node | Edge<Node>)[]
llm_nodes: (LLMNode | Edge<LLMNode>)[]
tools: ToolNode[]
```
