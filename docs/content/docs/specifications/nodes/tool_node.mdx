---
title: ToolNode
description: ToolNode is a node that holds a wrapper for a function calling by LLM.
---

## Specification

```yaml
# ToolNode

kind: ToolNode
name: string
description: LanguageBundle
func: string
arguments: ToolArgument[]
triggers: LanguageBundle[]
tool_prompt_template: Optional[LanguageBundle]
```

```yaml
# ToolArgument

name: string
type: string
description: Optional[LanguageBundle]
```
